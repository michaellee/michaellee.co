What is Jekyll?

Jekyll is a static-site generator written in Ruby. This means it lets you take simple text files (think markdown, you are using markdown right?) and make a website with it. Neat right? There's no databases, no updates and no viruses. Woohooo!

Why are you writing this book?

My personal site has been running on Jekyll for the past 3 years. I'm currently in the process of redesigning and rebuilding it. Along the way I'm documenting all the details of how I'm building my site with Jekyll.

In doing so I'm sharing all my tips and tricks that I've learned building sites with Jekyll so that you can build yours faster and awesomer!
